== Description ==

Hermit 音乐播放器：支持Html5+Flash 的音乐播放器。

详细介绍： http://mufeng.me/hermit-for-wordpress.html


== Installation ==

1. 上传 `Hermit`目录 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
